h1. JsDoc2 Template with Bootstrap, customised for modules.

h2. Abstract

This template allows for less cluttered visualisation of a project containing many classes which are ordered by module. The class list is collapsed
based on the module / class hierarchy.

h2. Based on

Fork of "JsDoc2 Template with Bootstrap":https://github.com/OrgaChem/JsDoc2-Template-Bootstrap .

See "Documentation page":http://orgachem.github.com/JsDoc2-Template-Bootstrap/ for more info.

h2. Install

bq. To produce output files you must use a template to format the generated documentation. Use the @-t@ option to specify which template you wish to use. 

bc. java -jar jsrun.jar app/run.js myscripts/ -t=templates/jsdoc

Show the official "documents":http://code.google.com/p/jsdoc-toolkit/wiki/CommandlineOptions#Using_a_Template , if you want more information.
